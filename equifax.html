EQUIFAX 2017 BREACH

The Vulnerability

Apache Struts CVE-2017-5638 OGNL injection exploited through content-type headers.

Attack Path

ACIS application → OGNL payload → code execution → lateral movement → 9,000+ queries.

Why It Happened

Missing patch mgmt, expired TLS cert, weak segmentation, blind monitoring.

Fixes & Prevention

Automated patching, WAF virtual patching, segmentation, DLP, certificate automation, SIEM.

Impact

147M+ records: SSN, DOB, addresses, payment info, license data.
